# Oopbronmodel

'n **Oopbronmodel** (Engels: open-source model) is 'n KI-model wat publiek beskikbaar gestel word, sodat enigeen dit kan aflaai, gebruik, bestudeer en dikwels aanpas.

## Eenvoudige verduideliking

Soos oopbron-sagteware (bv. Linux, Firefox), is oopbron-KI-modelle vrylik beskikbaar. Jy hoef nie te betaal om dit te gebruik nie, en jy kan sien hoe dit werk.

## Voordele

1. **Gratis** - geen lisensiekoste nie
2. **Privaatheid** - kan op jou eie rekenaar loop, data verlaat nooit jou stelsel nie
3. **Aanpasbaar** - kan verander word vir jou behoeftes
4. **Deursigtig** - jy kan sien hoe dit werk
5. **Gemeenskap** - baie mense dra by tot verbetering

## Bekende oopbronmodelle

- **Llama** (Meta) - kragtige algemene model
- **Mistral** - doeltreffende Franse model
- **Falcon** - uit die VAE
- **BLOOM** - meertalige model

## Oorwegings

- **Hulpbronne** - vereis dikwels kragtige hardeware om te loop
- **Ondersteuning** - geen amptelike kliëntediens nie
- **Kwaliteit wissel** - nie alles wat "oopbron" is, is goed nie
- **Lisensies verskil** - sommige beperk kommersiële gebruik

## Vir wie is dit?

- Ontwikkelaars wat wil eksperimenteer
- Maatskappye wat data privaat wil hou
- Navorsers wat wil verstaan hoe dit werk
- Enigeen wat nie afhanklik wil wees van groot tegnologiemaatskappye nie
