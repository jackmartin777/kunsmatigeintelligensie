# Prompt

'n **Prompt** is die teks wat jy aan 'n KI-stelsel gee om dit te vertel wat jy wil hê dit moet doen. Dit is jou instruksie aan die masjien.

## Eenvoudige verduideliking

Dink aan 'n prompt soos 'n bestelling by 'n restaurant. Hoe duideliker jy bestel, hoe meer waarskynlik kry jy wat jy wil hê.

## Voorbeelde

**Eenvoudige prompt:**
> "Skryf 'n gedig"

**Beter prompt:**
> "Skryf 'n kort gedig (4 lyne) oor die Karoo, in Afrikaans, met 'n nostalgiese toon"

## Hoekom dit belangrik is

Die kwaliteit van jou prompt bepaal grootliks die kwaliteit van die KI se antwoord. 'n Vae prompt gee 'n vae antwoord; 'n spesifieke prompt gee 'n spesifieke antwoord.

## Wenke vir goeie prompts

1. **Wees spesifiek** - sê presies wat jy wil
2. **Gee konteks** - verduidelik die agtergrond
3. **Spesifiseer formaat** - puntelys, paragraaf, tabel
4. **Noem beperkings** - lengte, toon, styl

## Verwante terme

Die kuns van goeie prompts skryf word "prompt engineering" genoem. Dit is 'n vaardigheid wat enige persoon kan leer.
