# Masjienleer

**Masjienleer** (Engels: Machine Learning) is 'n metode waar rekenaars leer om patrone te herken en voorspellings te maak deur voorbeelde te bestudeer, eerder as deur eksplisiete instruksies te volg.

## Eenvoudige verduideliking

Tradisionele programmering: Jy skryf reëls, die rekenaar volg dit.
Masjienleer: Jy gee voorbeelde, die rekenaar ontdek die reëls self.

## Hoe dit werk

1. **Data versamel** - baie voorbeelde van wat jy wil herken
2. **Model oplei** - die algoritme soek patrone in die data
3. **Evalueer** - toets hoe goed die model presteer
4. **Gebruik** - pas die model toe op nuwe data

## Tipes masjienleer

**Toesighouding (Supervised):**
- Jy gee voorbeelde mét antwoorde
- "Hier is fotos van katte en honde; leer die verskil"

**Sonder toesighouding (Unsupervised):**
- Jy gee data sonder antwoorde
- "Sorteer hierdie kliënte in groepe"

**Versterkingsleer (Reinforcement):**
- Die model leer deur beloning en straf
- "Speel hierdie speletjie; probeer wen"

## Voorbeelde

- Spam-deteksie in e-pos
- Gesigherkenning op jou foon
- Produk-aanbevelings op Amazon
- Kredietkaartbedrog-opsporing

## Beperkings

- Vereis baie data (dikwels duisende voorbeelde)
- Kan slegs leer wat in die data is
- "Garbage in, garbage out" - slegte data lei tot slegte modelle
